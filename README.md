# SortMe

SortMe est une application open source auto-hébergeable qui permet de trier automatiquement des emails grâce à des catégories et des règles écrites en langage naturel.

L’objectif est de connecter une boîte mail, définir des catégories personnalisées, ajouter des règles de tri compréhensibles par l’IA, puis laisser l’application classer les emails automatiquement.

SortMe peut fonctionner avec un LLM local pour garder les données sur son serveur, ou avec un provider cloud comme Mistral ou Gemini.
